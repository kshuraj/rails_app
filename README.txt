## Description of Zip file

### 1. house-prices folder
------------    

#### house-prices.Rmd
This file explains the R notebook for analyzing building on the train Data and predict test Data.


#### house-prices.html
This file explains the R notebook for analyzing building on the train Data and predict on test Data in the HTML format for browser view.

#### hp_train.csv
This file represents the input CSV and is used for training.

#### hp_test.csv
This file represents the input CSV and is used for testing. 


### 2. shiny_house folder
------------  

#### ui.R
This file represents the UI code for shiny UI application.

#### server.R
This file represents the server code for shiny-server applications.


#### house-price-Case.pdf
A complete explanation of the solution using Notebook and Shiny
